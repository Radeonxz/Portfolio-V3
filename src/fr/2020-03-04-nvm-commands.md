# Commandes NVM

#### liste toutes les versions de nœuds disponibles

```
nvm ls-remote
```

#### installer la version du nœud

```
nvm install x.x.x
```

#### sélectionner la version installée

```
nvm use node x.x.x
```

#### définir la version installée par défaut

```
nvm alias default x.x.x
```
